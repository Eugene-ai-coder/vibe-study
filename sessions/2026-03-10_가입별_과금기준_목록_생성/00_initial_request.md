# 종합 기능 개선 요구사항 (7건)

## 1. 개요 (Overview)
- **목적**: 가입 관리, 과금기준, Q&A, 공통코드 등 다수 기능의 개선 및 신규 화면 추가
- **개발 범위**: 기존 테이블 수정 2건, 신규 화면 2건, 기존 화면 개선 4건

## 2. 요구사항 목록

### 요구사항 1: 가입 테이블에 관리자ID 추가
- 가입(Subscription) 테이블에 관리자ID 컬럼 추가
- 관리자ID 입력 시 사용자(User) 팝업을 통해 조회하여 사용자ID를 선택/입력할 수 있도록 구현

### 요구사항 2: Main 페이지에 TODO 추가
- 메인 페이지에 TODO 영역 추가

### 요구사항 3: 과금기준 등록 진행상태 TODO 연동
- 과금기준이 등록되면, 등록진행상태가 "승인" 또는 "반려"가 아닌 경우 TODO에 표시

### 요구사항 4: 공통코드 관리 유효일자 기본값
- 공통코드 관리에서 유효시작일/유효종료일에 값이 있으면 해당 값 표시
- 값이 없으면 기본값으로 유효시작일=오늘, 유효종료일=9999/12/31 입력

### 요구사항 5: Q&A 공지사항 기능
- Q&A에 공지사항 등록 기능 추가
- 유효한 공지사항은 Q&A 목록 최상단에 항상 표시

### 요구사항 6: 가입별 과금기준 목록 조회 화면 신규
- 신규 화면: 가입별 과금기준 목록 조회
- 유효한 가입만 표시
- 유효한 과금기준이 없는 경우에도 가입정보는 목록에 표시

### 요구사항 7: 가입별 과금기준 목록에서 과금기준 화면 연동
- 가입별 과금기준 목록에서 행 클릭 시 과금기준 화면을 열어 해당 가입ID 기준으로 조회
